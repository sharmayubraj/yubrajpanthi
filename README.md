## üé® Customization Guide

### Personal Information

1. **Update HTML Content**:
   - Replace "Alex Johnson" with your name throughout `index.html`
   - Update the hero subtitle and description
   - Modify contact information (email, phone, location)
   - Update social media links

2. **Profile Images**:
   - Replace placeholder images with your professional photos
   - Update image alt texts for accessibility
   - Ensure images are optimized for web (WebP format recommended)

### Content Customization

1. **About Section**:
   - Update background story and experience
   - Modify statistics (projects completed, years experience, etc.)
   - Add your personal achievements

2. **Skills Section**:
   - Update skill names and proficiency levels
   - Modify progress bar percentages in both HTML and CSS
   - Add or remove skill badges as needed
   - Update technology categories

3. **Projects Section**:
   - Replace sample projects with your actual work
   - Update project images, titles, and descriptions
   - Add real project links (live demo and GitHub)
   - Modify technology tags for each project

### Styling Customization

1. **Color Scheme**:
   ```css
   :root {
     --primary-color: #6366f1;    /* Main brand color */
     --accent-color: #06d6a0;     /* Accent color */
     --text-primary: #1f2937;     /* Main text color */
     /* Update other color variables as needed */
   }
   ```

2. **Typography**:
   - Change Google Fonts imports in HTML head
   - Update font family variables in CSS
   - Adjust font sizes and weights

3. **Layout**:
   - Modify spacing variables
   - Adjust container max-width
   - Update grid layouts and breakpoints

### Adding New Sections

1. **HTML Structure**:
   ```html
   <section class="new-section" id="new-section">
     <div class="container">
       <div class="section-header" data-aos="fade-up">
         <h2 class="section-title">Section Title</h2>
         <p class="section-subtitle">Section description</p>
       </div>
       <!-- Section content -->
     </div>
   </section>
   ```

2. **Navigation**:
   - Add new link to navigation menu
   - Update JavaScript for smooth scrolling

3. **Styling**:
   - Add CSS rules following existing patterns
   - Include responsive breakpoints
